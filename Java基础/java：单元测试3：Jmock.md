# java：单元测试3：Jmockit

* JMockit是用来帮助开发人员编写测试程序的一组工具和API，完全基于java 5 ee的java.lang.instrument包开发
* 两种mock方式：
  * Behavior-oriented（Expectations & Verifications）基于行为
  * State-oriented（MockUp<GenericType）基于状态
